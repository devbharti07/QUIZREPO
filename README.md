# PulseQuiz Arena

Responsive quiz game built with plain HTML, CSS, and JavaScript.

## Features

- Multiple-choice rounds with 4 options per question
- Timed questions and a visible quiz progress bar
- Easy, medium, and hard difficulty modes
- Shuffled questions and shuffled answers every run
- Final score, accuracy summary, and question review
- Daily streak tracking with once-per-day reward claims
- Streak milestone bonuses on day 3, 7, and 15
- Local leaderboard and persistent `localStorage` data
- Resume support for an unfinished quiz
- Sound effects for correct, wrong, and timed-out answers

## Run

Open `index.html` in a browser.

No build step or package install is required.
